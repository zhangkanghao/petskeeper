# 简介
一个宠物app平台，包含个人中心，交流中心，和交易中心的一个移动应用后台。


# 功能
## 个人区：
#### 个人信息（初步确定，待修改）：
- 个人基本信息：昵称、邮箱（激活）、性别、介绍、住址、头像
- 个人交互信息：关注、粉丝、被赞、点赞
- 个人高级信息：我的发布、我的收藏、草稿箱、我的宠物    （一对多）

## 动态区：
#### 帖子的罗列
- 经验贴：分享养宠物经验
- 求助帖：对宠物出现问题进行求助。（可以拓展到寄养，得加上身份认证...）
- 科普贴：科普铲屎官界内常用知识
## 消息区：

- 系统消息：
- 个人消息：和互动用户聊天（待考虑）
## 首页（动态）：
对于点赞数和浏览数多的帖子进行加精并放到首页进行推荐
